mauibuild-make-builddisks(1)
============================

Name
----
mauibuild-make-builddisks - Build uncompressed disk images.

Synopsis
--------
'mauibuild make -n builddisks'

Description
-----------
Build uncompressed disk images for the runtime target, incrementally updated with 'ostree'.

mauibuild-checkout(1)
=====================

Name
----
mauibuild-checkout - Check out one or more git repositories

Synopsis
--------
'mauibuild checkout' [<options>] <component>...

Description
-----------
Checks out one or more git repositories.

Options
-------
*\--overwrite*::
	Overwrite.

*\--patches-path* <path>::
	Specify the path where patches are saved.

*\--metadata-path* <path>::
	Specify the path where metadata is located.

*\--workdir* <path>::
	Specify an alternative working directory.

*\--snapshot* <path>::
	Specify an alternative snapshot file.

*\--checkoutdir* <path>::
	Specify where the git repositories will be checked out.

*\--clean*::
	Clean.

<component>...::
	One or more components whose git repositories will be checked out.

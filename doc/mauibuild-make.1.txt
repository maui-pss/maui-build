mauibuild-make(1)
=================

Name
----
mauibuild-make - Execute tasks

Synopsis
--------
'mauibuild make' [<options>] [-- <task arguments>...]

Description
-----------
Execute tasks.

Options
-------
*-l, \--list*::
	List all available tasks and exit.

*-n, \--task-name*::
	Specify name of the task to execute.

*-u, \--only*::
	Don't process tasks after this.

*\--task-help*::
	Show help message for the specified task and exit.

*-x, \--skip <name>*::
	Skip the task <name>.

*-- <task arguments>...*::
	Arguments for the specified task.

Tasks
-----
*resolve*::
	Expand git revisions in source to exact targets.

*build*::
	Build multiple components and generate trees.

*bdiff*::
	Report differences between builds.

*builddisks*::
	Generate uncompressed disk images updated with ostree.

*zdisks*::
	Generate compressed disk images.

*buildlive*::
	Generate live images.

mauibuild-make-build(1)
=======================

Name
----
mauibuild-make-build - Build multiple components and generate trees.

Synopsis
--------
'mauibuild make -n build' [-- <components>...]

Description
-----------
Builds all the components, starting from the base system and generates trees.

At the end of the process a binary snapshot is generated in *$workdir/results/build*.

Options
-------
*<components>...*::
	Force rebuild of these components.

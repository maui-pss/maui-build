mauibuild-make-bdiff(1)
=======================

Name
----
mauibuild-make-bdiff - Report differences between builds.

Synopsis
--------
'mauibuild make -n bdiff'

Description
-----------
Reports differences between builds, difference is calculated comparing binary snapshots.

At the end of the process a JSON report is available under *$workdir/results/bdiff*.
